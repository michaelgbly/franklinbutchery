## Internal Terms

| Term | Meaning |
|------|---------|
| **The five pillars** | Butcher case, Seafood, Sandwiches, Beer & wine, Meals To Go |
| **Meals To Go** | Refrigerated, chef-driven prepared meals for reheating at home; a major pillar |
| **Mother sauces** | House-made base sauces sold as retail add-ons |
| **Store within a store** | The Roper Field / Lavender Row merch section in the front of the shop |
| **Roper Field** | Handmade leather goods brand, made in Franklin, sold in-store |
| **Lavender Row** | Lavender products, sold under the Roper Field umbrella |
| **Cut room** | Fully glass-walled room where animals are broken down and dry-aged; visible to guests |
| **Salt wall** | Himalayan salt wall in the cut room; aids dry-aging via air ionization and humidity |
| **Elevated experience** | Preferred brand term (not "upscale" or "high-end") |
| **Supper club** | Future dinner events; slow launch; announced via Instagram/email |
| **Private dining** | Rentable dining room for events; inquiry-based, not reservation-based |
| **Edge** | Local comparable venue referenced for event booking process |
| **Perrin** / **Perrin's** | Local restaurant doing Thu/Fri/Sat dinners; referenced as dinner model |
| **Toast** | POS system (mentioned but private dining does NOT go through Toast) |

|---------|--------|-----------|
| Beef | Bear Creek Farm + Simpsons | Pasture-raised, grain-finished, traceable to individual cow, ~2.5 cows/week |
| Grass-fed beef | Separate source | Half cow of fully grass-fed, distinct from grain-finished Bear Creek |
| Pork | Bear Creek Farm | Same pasture-raised standards |
| Poultry | Springer Mountain | Tennessee/Georgia, premium, co-op style |
| Seafood | Maine | Fresh, not frozen (except possibly shrimp) |
| Spices | Various | All organic |
| Cures/bacon | In-house | All blended and processed on-site |

## Brand Language Rules (From Owners)

| Do Say | Don't Say |
|--------|-----------|
| "Elevated experience" | "Upscale" (only as backup) |
| "Ionization of the air for dry aging" | "Bacteria" (re: dry aging process) |
| "Warm industrial" | "High-end" (too sterile) |
| "Traceable" / "we know where it comes from" | Overclaims like "organic" or "certified" unless confirmed |
| "Pasture-raised, grain-finished" | "Corn-fed" or "grain-fed" (it's a nutritionist-blended feed) |
